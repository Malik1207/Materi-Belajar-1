TUGAS STRING METHOD

1. ChartAt untuk mengembalikan karakter pada indeks tertentu dalam sebuah string.

2. ChartCodeAt untuk mengembalikan karakter pada indeks tertentu dengan kode utf-16 dalam sebuah string.

3. Concat untuk menggabungkan dua atau lebih sebuah string.

4. Includes untuk mengembalikan sebuah string dalam sebuah string jika benar akan muncul TRUE jika salah FALSE

5. EndWith Includes untuk mengembalikan sebuah string terakhir dalam sebuah string jika benar akan muncul TRUE jika salah FALSE

6. IndexOf untuk mengembalikan indeks pertama dari nilai tertentu dalam sebuah string.

7. LastIndexOf untuk mengembalikan indeks dari kejadian terakhir nilai tertentu dalam sebuah string.

8. Match untuk mengembalikan sesuatu yang cocok dalam sebuah string

9. MatchAll untuk mengembalikan semua sesuatu yang cocok dalam sebuah string.

10. Repeat untuk mengembalikan string dengan jumlah salinan dari string.

11. Replace untuk mengembalikan string baru.

Berakit-rakit ke hulu berenang-renang ke tepian

12. ReplaceAll untuk mengembalikan string baru dengan semua kecocokan pola diganti dengan pengganti.

Bersakit-sakit dahulu bersenang-senang kemudian

Bersakit-sakit dahulu bersenang-senang kemudian

13. Search untuk pencarian string untuk nilai.

14. Slice untuk ekstrak bagian dari string.

15. Split untuk membagi string menjadi array substring.

16. StarWith untuk mengembalikan sebuah string pertama dalam sebuah string jika benar akan muncul TRUE jika salah FALSE.

17. Substr untuk mengekstrak bagian string.

18. ToLowerCase untuk mengkonversi string ke huruf kecil huruf.

19. ToUpperCase untuk mengkonversi string ke huruf kecil besar.

20. ToString untuk mengkorversi object string jadi string.

21. ToValueOf untuk mengkorversi object string jadi string.
